Production build note
---------------------
A Linux production build could not finish in this sandbox because the available
saved dependency folder contained the Windows SWC binary and the package registry
returned HTTP 503 while Next.js attempted to download the Linux SWC binary.
Run `npm ci && npm run build` on the deployment machine or Vercel. The release ZIP
excludes node_modules and .next so the correct Next.js 16.2.11 platform dependencies
will be installed from package-lock.json.
